# 智能出题功能实施计划

## 目标
在智能出题界面，用户输入知识点并点击生成题目后，正确向后端发送请求。

## 后端接口信息
- **URL**: `localhost:8000/agent/question/generate_question`
- **方法**: `POST`
- **请求体格式**: 
```json
{
  "question": "明朝历史",
  "question_type": 0,
  "question_count": 1
}
```

## 字段说明
- `question`: 用户输入的知识点
- `question_type`: 题型选择
  - 0: 选择题
  - 1: 填空题
  - 2: 判断题
  - 3: 主观题
- `question_count`: 题目数量 (1-20)

- 使用 fetch API 发送 POST 请求

- 保持与现有代码风格一致

## 预期结果

1. 向后端发送正确的请求
2. 显示生成进度
3. 成功后显示提示信息
4. 如遇认证错误，弹出登录提示框
